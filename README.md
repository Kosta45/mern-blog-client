# MERN Easy Blog Client

Frontend client for a full-stack blog application built with the MERN stack.

This project was created as a hands-on learning project while studying full-stack development with React, Node.js, Express and MongoDB.

The application allows users to register, log in, create posts, upload images and interact with blog content through a modern React interface.

---

# Features

- User registration and login
- JWT authentication
- Create, edit and delete posts
- Upload post images
- View posts and article details
- Responsive UI
- State management with Redux Toolkit
- API communication with Axios

## Planned Features

- Comments system
- Post filtering and sorting
- User subscriptions
- Profile customization

# Installation

## Clone repository

```bash
git clone https://github.com/Kosta45/mern-easy-blog-client.git
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

Application will be available at:

```bash
http://localhost:5173
```
